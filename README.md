# Книжка контактов

### Функционал:
Данное приложение реализует функционал записной книжки, и позволяет:
- Добавлять контакты
- Удалять контакты
- редактировать контакты

### Установка:
Клонирование проекта:
```
git clone https://github.com/KlimentFis/Contacts.git
```
Переход в папку проекта:
```
cd Contacts
```
